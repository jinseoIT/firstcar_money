## 첫차 Money
개발 기간 2021.09.13 ~ 2021.09.17

---

## 🎈 프로젝트 소개

### 사회초년생들을 위한 첫차 구매 프로젝트 첫 차 Money? 나의 연봉에 맞는 자동차부터 다양한 국산 차들까지 새차가 부담이 된다면 중고차 사이트도 있으니 나에게 맞는 자동차를 찾아 보세요.


## 📄 구성

- 메인 (차량 리스트, youtube)
- 로그인
- 회원가입
- 차량 리스트 / 검색
- 댓글 - 추가 , 읽기 , 수정, 삭제,



## 👨‍👩‍👧‍👧팀 구성

- 고규식
- 김기태
- 양진성
- 배수인


## 🔨 구현 기능 

- 로그인 회원가입 
- 데이터 크롤링
- CRUD + Auth (Auth 된 유저만 UD 가능)
- 스크롤바
- 반응형 페이지 
- 데이터 정렬
